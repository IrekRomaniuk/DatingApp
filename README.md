# DatingApp


DatingApp.API
```
dotnet tool install --global dotnet-ef --version 3.0.0
dotnet restore
dotnet ef migrations add InitialCreate
dotnet ef database update
dotnet watch run
http://localhost:5000/api/values
```


DatingApp-SPA
```
brew install node
npm install -g @angular/cli
npm install
ng serve
http://localhost:4200/
```
